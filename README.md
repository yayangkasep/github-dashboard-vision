# GitVision 🚀

A stunning, modern GitHub profile dashboard with real-time analytics, repository insights, contribution graphs, and language breakdowns.

## Features

- **Modern UI**: Glassmorphism, dark mode, and smooth animations.
- **Real-time Data**: Fetches live data from the GitHub API.
- **Responsive Design**: Built with mobile-first principles.
- **Interactive Elements**: Hover effects, tooltips, and smooth scrolling.
- **Particle Effects**: Subtle background particles for a premium feel.

## Tech Stack

- **HTML5**
- **CSS3** (Custom Properties, Flexbox, Grid)
- **JavaScript (Vanilla)**

## Installation

1.  Clone the repository:
    ```bash
    git clone <repository-url>
    cd github-dashboard-vision
    ```

2.  Open `index.html` in your browser.

## Usage

1.  Enter a GitHub username in the search bar.
2.  Click **Explore** or press **Enter**.
3.  View the detailed dashboard with stats, repos, and language breakdown.
